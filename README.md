# Mortgage-Plan

This is a Java program to calculate and display mortgage using the provided formula.
This was a code test for a traineeship.
I programmed and built the program on a linux laptop.


How to build and run the application:

1. Download Apache Maven from: https://maven.apache.org/download.cgi
2. Follow the install instructions in the link or in Mavens readMe file: https://maven.apache.org/install.html
3. place the prospects.txt in your home directory (C:\users\CurrentUser on windows, /home/CurrentUser/ on Linux and /Users/CurrentUser on Mac)
4. open a command prompt or terminal and cd to the Mortgage-Plan directory
5. run "mvn package" to compile the program into a .jar file
6. run "java -cp target/Mortgage-Plan-1.0-SNAPSHOT.jar ax.ha.it.mortgage.Main" to run the program
